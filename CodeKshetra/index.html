About Code-क्षेत्र

Welcome to Code-क्षेत्र, a thrilling 36-hour hackathon hosted at JIMS Rohini Sector-5 in February 2024. More than just a competition, it's an immersive experience filled with innovation, exciting prizes, swags, and a lot of fun. It's your opportunity to connect with experienced mentors and judges.

Organized by Geek Room, a vibrant community dedicated to enhancing coding skills, started as an open community for solving tech-related queries and participating in college competitions. In a short span, it has grown to include 6000+ participants from colleges across India. Now, launching chapters in different colleges, we're excited to have it at JIMS!

Join us on this exciting coding journey and become a part of Geek Room's JIMS Chapter. Connect with us on Geek Room's JIMS Chapter.

F.A.Q

  • What is Code Kshetra?

    Code Kshetra is an annual hackathon organized by JIMS Rohini Sector 5. It provides a platform for participants to showcase their coding and problem-solving skills over a 36-hour intense coding competition.

  • Who can participate in Code Kshetra?

    Code Kshetra is open to students and coding enthusiasts from any educational background. Whether you're a beginner or an experienced coder, everyone is welcome to participate and showcase their skills.

  • When and where will Code Kshetra 2024 take place?

    Code Kshetra 2024 will be held in February at JIMS Rohini Sector 5. Please refer to the official website or contact the organizers for specific dates and venue details.

  • How can I register for Code Kshetra 2024?

    To register for Code Kshetra 2024, visit the official website and follow the registration instructions. Complete the registration form and make sure to adhere to the specified deadlines.

  • Are there any prerequisites for participating in Code Kshetra?

    Code Kshetra is open to participants with varying skill levels. While there are no strict prerequisites, having a basic understanding of programming concepts and algorithms can enhance your experience during the hackathon.

  • What are the prizes for the winners of Code Kshetra?

    Code Kshetra offers exciting prizes for the winners, including cash rewards, certificates, and potential opportunities for internships or job placements. The exact details of the prizes will be announced closer to the event.

  • Can I participate in a team or as an individual?

    Code Kshetra allows both team and individual participation. You can form a team with fellow participants or register as an individual. Make sure to check the team size and composition guidelines provided in the event rules.
